# 🚀 PathFinderPro: Shortest Route Calculator

## 📌 Project Overview
PathFinderPro is a route optimization system that calculates the shortest distance between cities using **Dijkstra’s Algorithm, Bellman-Ford Algorithm, and the Travelling Salesman Problem (TSP)**. This project is designed to help users find the most efficient routes for navigation, logistics, and transportation.

## 🛠 Features
- ✅ **Dijkstra’s Algorithm** – Finds the shortest path between two cities.
- ✅ **Bellman-Ford Algorithm** – Works with graphs containing negative weights.
- ✅ **Travelling Salesman Problem (TSP)** – Optimizes routes for multiple cities.
- ✅ **Graph Visualization** – Displays city-to-city connections interactively.
- ✅ **User Input Support** – Enter city names, distances, and view optimal paths.

## 📌 Technologies Used
- **C++** – Core implementation of algorithms.
- **Graph Data Structures** – Efficiently storing and processing city networks.
- **Graphics Library** –  Graphical representation of routes.
